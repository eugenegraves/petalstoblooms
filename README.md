# Petals to Blooms

A modern React website for Petals to Blooms - Unique Coastal Floral Arrangements.

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Modern CSS
