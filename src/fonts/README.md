# Fonts Directory

Please add the following custom fonts to this directory when available:

1. **PalmBeachStyle-Regular.woff2** / **PalmBeachStyle-Regular.woff**
   - Used for main title "PETALS TO BLOOMS" (36pt)

2. **AvenirNextCondensed-Bold.woff2** / **AvenirNextCondensed-Bold.woff**
   - Used for subtitle (10pt)

3. **PinyanScript-Regular.woff2** / **PinyanScript-Regular.woff**
   - Used for call-to-action text "Ask About Our Concierge Service" (24pt)

Note: Helvetica is a system font and doesn't need to be added here.

The CSS is already configured to use these fonts. Simply add the font files and they will be automatically loaded.
